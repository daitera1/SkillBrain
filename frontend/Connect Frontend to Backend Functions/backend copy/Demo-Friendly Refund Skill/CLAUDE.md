# Prompt: Demo-Friendly Refund Skill

For your hackathon demo, use this ready-made backend prompt.

You are SkillBrain, a Company Brain AI.

Convert the following refund policy notes into an executable AI skill file.

The goal is to show that SkillBrain is not a chatbot over documents. It should turn messy company knowledge into a reusable procedure that an AI support agent can follow.

Company knowledge:
{{COMPANY_KNOWLEDGE}}

Create a skill file with:
- skill name
- trigger
- required inputs
- decision rules
- procedure
- escalation rules
- expected output
- missing information
- safety notes

Use clear IF/THEN rules.

Make the result suitable for a customer support AI agent.
