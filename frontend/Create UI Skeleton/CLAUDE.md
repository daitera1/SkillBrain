# Create the UI Skeleton

You are the frontend engineer for a 5-hour hackathon MVP called SkillBrain.

SkillBrain is a Company Brain product. It turns scattered company knowledge into executable AI skill files for AI agents.

Build the frontend UI for a Next.js + React + TypeScript + Tailwind app.

The app should look like a polished AI SaaS demo page.

Create one main page with these sections:

1. Hero section
- Product name: SkillBrain
- Tagline: Turn scattered company knowledge into executable AI skills.
- Short explanation: Not a document chatbot — a living map of how company work gets done.

2. Three-step explanation cards
- Input fragmented knowledge
- Generate executable skill
- Test with real scenario

3. Main workspace with two columns
Left column:
- Workflow type dropdown
- Company knowledge textarea
- Load sample button
- Generate Skill File button

Right column:
- Generated skill file output panel
- Copy skill button

4. Scenario testing section
Left:
- Scenario textarea
- Run Skill button

Right:
- Agent decision output panel with:
  - Decision
  - Reasoning
  - Next action
  - Escalation status
  - Message draft

5. Why this is Company Brain section
Cards:
- Not document search
- Living workflow map
- Agent-ready skills

Requirements:
- Use clean dark theme.
- Use rounded cards, spacing, clear typography.
- Make it readable in a demo video.
- Use local React state only.
- Do not add authentication, database, PDF upload, or complex integrations.
- Use placeholder/mock functions if backend is not ready.
