# Improve Layout for 90-Second Demo

Improve the SkillBrain frontend for a 90-second hackathon demo.

Make the user flow extremely obvious:

Step 1: Paste messy company knowledge
Step 2: Generate executable skill file
Step 3: Test the skill on a real scenario
Step 4: Show agent decision

Add visible step labels above each major section.

Make the generated skill output look like a code/document viewer.

Make the agent decision output look like a decision card with:
- large decision title
- reasoning bullets
- escalation badge
- next action box
- message draft box

Keep the UI simple and do not add unnecessary features.

The page should be impressive but still easy to understand at a glance.
